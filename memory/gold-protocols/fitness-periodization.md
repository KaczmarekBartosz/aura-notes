# PERIODYZACJA I PROGRESJA

*Systematyczne podejście do długoterminowego rozwoju siły i hipertrofii*

---

## 1. ZASADY PROGRESJI PRZECIĄŻENIA

### Progressive Overload - Fundament

**Definicja:** Systematyczne zwiększanie wymagań treningowych w czasie, wymuszające adaptację.

**6 głównych zmiennych progresji (w kolejności od najważniejszej):**

| Zmienna | Przykład zastosowania | Częstotliwość |
|---------|----------------------|---------------|
| **1. Siła (load)** | 100kg → 102.5kg → 105kg | Co sesję/tydzień |
| **2. Objętość (sets)** | 3 serie → 4 serie → 5 serii | Co 1-2 tygodnie |
| **3. Powtórzenia (reps)** | 8 reps → 9 reps → 10 reps | W obrębie zakresu |
| **4. Częstotliwość (frequency)** | 1x/tydzień → 2x/tydzień | Co mesocykl |
| **5. Gęstość (density)** | 40 min → 35 min (mniej przerwy) | Stopniowo |
| **6. Ekscycentryczne (tempo)** | 3s → 4s oporu | Okresowo |

**Zasada 80% Rule:** Dąż do progresji w 80% sesji. Jeśli nie masz progresji w ≥50% serii przez 2 tygodnie → czas na zmianę stymulu.

---

### Double Progression - Dla Pośrednich

**Metoda:** Najszybsza i najprostsza ścieżka progresji.

**Protokół:**
```
Ćwiczenie: Wyciskanie ławeczkowe
Zakres celowy: 8-12 powtórzeń
Obciążenie startowe: 80kg (taki, że robisz bezpiecznie 8, ale 12 jest wyzwaniem)

Tydzień 1: 80kg × 8, 8, 7 reps (RIR 2-3)
Tydzień 2: 80kg × 9, 9, 8 reps (RIR 1-2)
Tydzień 3: 80kg × 10, 10, 9 reps (RIR 0-1)
Tydzień 4: 80kg × 12, 11, 11 reps (OSIĄGNIĘTO górną granicę)
Tydzień 5: 82.5kg × 8, 8, 7 reps (RESET powtórzeń, +2.5% load)
```

**Kluczowe zasady:**
- **Zwiększ obciążenie** tylko gdy osiągasz **górną granicę zakresu** we wszystkich seriach roboczych
- **Zwiększaj powtórzenia** między treningami, jeśli obciążenie jest stałe
- **Minimum increment:** 1.25-2.5% dla dużych ćwiczeń (squat, deadlift), 2.5-5% dla izolacji

**Ważne:** Jeśli spadasz o ≥2 powtórzenia poniżej celu przez 2 sesje → redukcja obciążenia o 5-10%.

---

### RIR (Reps In Reserve) i RPE (Rate of Perceived Exertion)

**Skala RIR - dystans do upadku:**

| RIR | Opis | Zastosowanie |
|-----|------|--------------|
| **0** | True failure (techniczne) | Rzadko, głównie izolacja |
| **1** | 1 powtórzenie w zapasie | Hipertrofia główna |
| **2** | 2 powtórzenia w zapasie | **SWEET SPOT** dla większości |
| **3** | 3 powtórzenia w zapasie | Deload, ćwiczenia techniczne |
| **4+** | Łatwe, za mało stymulu | Rozgrzewka, aktywacja |

**Skala RPE - subiektywna intensywność (1-10):**
```
RPE 10 = Maksymalny wysiłek (0 RIR)
RPE 9  = Bardzo ciężki (1 RIR)
RPE 8  = Ciężki (2 RIR)
RPE 7  = Umiarkowanie ciężki (3 RIR)
RPE 6  = Umiarkowany (4 RIR)
RPE 5  = Lekki (5+ RIR)
```

**Protokół autoregulacji RIR (Israetel):**

| Faza mesocyklu | Cel treningowy | RIR celowy | RPE |
|----------------|----------------|------------|-----|
| Akumulacja | Hipertrofia | 2-3 RIR | 7-8 |
| Intensyfikacja | Siła + hipertrofia | 1-2 RIR | 8-9 |
| Realizacja | Maksymalizacja | 0-1 RIR | 9-10 |
| Deload | Regeneracja | 4-5 RIR | 5-6 |

**Korekty auto-regulacji:**
- Sen < 6h → +1 RIR (zmniejsz intensywność)
- Stres życiowy wysoki → +1 RIR
- Ciśnienie spoczynkowe elevated → +1-2 RIR
- Motywacja niska → +1 RIR

## 2. MODELE PERIODYZACJI

### Porównanie: Liniowa vs Blokowa vs Undulating

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TRZY GŁÓWNE MODELE                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  LINEAR PROGRESSION (LP)                                            │
│  ────────────────────────                                          │
│  • Constant load increase, decreasing reps                          │
│  • Najlepsze dla: BEGINNER                                         │
│  • Przykład: 5x5 @ 60kg → 62.5kg → 65kg                             │
│  • Limit: kiedy progresja spowalnia do <1kg/tydzień                 │
│                                                                     │
│  BLOCK PERIODIZATION                                                │
│  ─────────────────────                                              │
│  • Fazy skupione na konkretnym celu (4-6 tyg każda)                 │
│  • Blok 1: Hypertrophy (wysoki vol, średni intensity)              │
│  • Blok 2: Strength (średni vol, wysoki intensity)                  │
│  • Blok 3: Peaking (niski vol, maksymalny intensity)               │
│  • Najlepsze dla: INTERMEDIATE → ADVANCED                           │
│                                                                     │
│  DAILY UNDULATING PERIODIZATION (DUP)                               │
│  ──────────────────────────────────                                 │
│  • Różne stimuli w obrębie tygodnia                                │
│  • Pon: Siła (3x5, 85%)                                             │
│  • Śro: Hipertrofia (3x10, 70%)                                     │
│  • Pią: Power (5x3, 80%, eksplozywność)                             │
│  • Najlepsze dla: INTERMEDIATE                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

---

### Liniowa Progresja (Linear Progression)

**Wzór:**
```
Tydzień 1: 3x8 @ 60kg (RIR 3)
Tydzień 2: 3x8 @ 62.5kg (RIR 2-3)
Tydzień 3: 3x8 @ 65kg (RIR 2)
Tydzień 4: 3x8 @ 67.5kg (RIR 1-2)
Tydzień 5: 3x8 @ 70kg (RIR 1) → DELOAD
Tydzień 6: Reset + 2.5kg → 3x8 @ 65kg
```

**Znaki PF (Plateau Failure) - czas zmienić:**
1. Nie możesz dodać 1.25-2.5kg przez 2 tygodnie
2. RIR stale rośnie (nasycony stymulem)
3. Motywacja spada, bo trening jest "za łatwy"

**Gdy PF występuje:** Przejdź na Double Progression lub Block.

---

### Blokowa Periodyzacja (Block Periodization)

**Czas trwania bloku: 4-6 tygodni** (potem nadadaptacja = spadek zysków)

**Struktura typowego mesocyklu blokowego:**

#### BLOK 1: AKUMULACJA (Hypertrophy Focus)
**Cel:** Budowa fundamentu mięśniowego
- **Objętość:** WYSOKA (12-20 setów/partia/tydzień)
- **Intensywność:** ŚREDNIA (RIR 2-3, 65-75% 1RM)
- **Powtórzenia:** 8-15
- **Czas trwania:** 4-6 tygodni

**Przykładowy mikroskładnik:**
```
PUSH ( Pon, Czw)
├─ Bench press: 4x10 (70%)
├─ OHP: 3x10 (67%)
├─ Dips: 3x12
└─ Lateral raises: 4x15

PULL ( Wt, Pt)
├─ Pull-ups: 4x10
├─ Barbell row: 4x10
├─ Face pulls: 3x15
└─ Biceps curl: 3x12

LEGS ( Śr, Sb)
├─ Squat: 4x10 (72%)
├─ RDL: 4x10
├─ Leg press: 3x12
└─ Calf raises: 4x15
```

#### BLOK 2: INTENSYFIKACJA (Strength + Hypertrophy)
**Cel:** Przenoszenie nowej masy do siły
- **Objętość:** UMIARKOWANA (8-12 setów/partia/tydzień)
- **Intensywność:** WYSOKA (RIR 1-2, 80-88% 1RM)
- **Powtórzenia:** 4-8
- **Czas trwania:** 4-6 tygodni

**Przykładowy mikroskładnik:**
```
PUSH ( Pon, Czw)
├─ Bench press: 4x6 (82%)
├─ OHP: 3x6 (80%)
├─ Incline DB press: 3x8
└─ Lateral raises: 3x12

PULL ( Wt, Pt)
├─ Weighted pull-ups: 4x6
├─ Barbell row: 4x6 (80%)
├─ Face pulls: 3x12
└─ Biceps curl: 3x10

LEGS ( Śr, Sb)
├─ Squat: 4x6 (83%)
├─ Front squat: 3x6 (78%)
├─ RDL: 3x8
└─ Leg curl: 3x10
```

#### BLOK 3: REALIZACJA (Peaking)
**Cel:** Maksymalizacja siły/hipertrofii
- **Objętość:** NISKA (4-8 setów/partia/tydzień)
- **Intensywność:** BARDZO WYSOKA (RIR 0-1, 88-95% 1RM)
- **Powtórzenia:** 1-5
- **Czas trwania:** 3-4 tygodnie

**Przykładowy mikroskładnik:**
```
PUSH ( Pon, Czw)
├─ Bench press: 3x4 (90%)
├─ OHP: 2x4 (88%)
├─ Dips: 3x8
└─ Lateral raises: 3x10

PULL ( Wt, Pt)
├─ Weighted pull-ups: 3x4 (90%)
├─ Barbell row: 3x5 (85%)
├─ Face pulls: 2x15
└─ Biceps curl: 2x10

LEGS ( Śr, Sb)
├─ Squat: 3x4 (92%)
├─ Deadlift: 2x3 (90%)
└─ RDL: 3x6
```

#### BLOK 4: DELOAD (Regeneracja)
**Cel:** Reszte CSS (cumulative stress), nadadaptację
- **Objętość:** 50% z bloku 3
- **Intensywność:** 70-80% z bloku 3 (ale RIR 4-5)
- **Czas trwania:** 1 tydzień

---

### Daily Undulating Periodization (DUP)

**Struktura tygodniowa:**

| Dzień | Fokus | Zakres % | Powtórzenia | RIR |
|-------|-------|----------|-------------|-----|
| Pon | Siła | 80-88% | 3-6 | 2-3 |
| Wt | Hipertrofia | 67-75% | 8-12 | 2-3 |
| Śr | Rest/Active recovery | - | - | - |
| Czw | Moc (Power) | 30-50% | 3-5 (eksplozywnie) | 3-4 |
| Pt | Hipertrofia | 70-78% | 10-15 | 2-3 |
| Sb | Metabolic | 60-70% | 15-20 (drop sets) | 2-3 |
| Nd | Rest | - | - | - |

**Kiedy DUP działa najlepiej:**
- Użytkownik ma 3+ lat treningu
- Plateau na LP
- Chce zróżnicowanych bodźców
- Ma czas na 3-4 sesje/tydzień

## 3. DELOAD PROTOCOLS

**Definicja:** Planowana redukcja stresu treningowego dla regeneracji i nadadaptacji.

### Kiedy Deload? (3 metody)

#### Metoda 1: Time-Based (dla beginner/intermediate)
```
Co 4-6 tygodni intensywnego treningu → 1 tydzień deload
```

#### Metoda 2: Fatigue-Based (zaawansowani)
**Znaki potrzeb deload:**
- Spadek siły w ostatnich 2 sesjach o ≥5%
- RIR stale rośnie mimo stałego obciążenia (5 RIR tam gdzie były 2)
- CIśnienie spoczynkowe ↑ 5+ mmHg przez 3 dni
- Sen niska jakość + motywacja ↓
- Bóle stawowe (nie muscle soreness)

#### Metoda 3: Auto-Regulated (Israetel)
**Skala zmęczenia:**
```
0-2 = Brak zmęczenia → treń normalnie
3-4 = Lekkie zmęczenie → utrzymaj, nie zwiększaj
5-6 = Umiarkowane → rozważ deload w tym tygodniu
7-8 = Wysokie → DELOAD TERAZ
9-10 = Przetrenowanie → 2 tygodnie deload
```

### Jak Deload? (3 protokoły)

#### Protokół A: Volume Reduction (50% rule)
- **Objętość:** 50% normalnej (np. 3 serie → 2 serie)
- **Intensywność:** Utrzymaj (lub -10% load jeśli bardzo zmęczony)
- **Częstotliwość:** Utrzymaj
- **RIR:** 4-5 (poczujesz "lekko")

**Przykład:**
```
Normalne: 4x8 @ 80kg (RIR 2)
Deload:   2x6 @ 80kg (RIR 4) → 50% sets, 75% reps
```

#### Protokół B: Intensity Reduction
- **Objętość:** Utrzymaj
- **Intensywność:** -15 do -20% 1RM
- **RIR:** 4-5
- **Zastosowanie:** Głównie gdy stawy bole

**Przykład:**
```
Normalne: 4x8 @ 80kg (RIR 2)
Deload:   4x8 @ 65kg (RIR 4) → -19% load
```

#### Protokół C: Frequency Reduction
- **Objętość/intensywność:** Utrzymaj w sesjach
- **Częstotliwość:** -1 sesja/tydzień
- **Zastosowanie:** Gdy czas jest ograniczony

**Przykład:**
```
Normalne: 4 sesje (Pon, Wt, Czw, Pt)
Deload:   3 sesje (Wt, Czw, Pt) lub (Pon, Śr, Pt)
```

### Co robić w deload?

**Zalecane:**
- Lekki cardio (2-3x/tydzień, 20-30 min, 60-70% HRmax)
- Stretching/mobility (codziennie 10-15 min)
- Sen: 8+ godzin (priorytet!)
- Nawodnienie: 3-4L wody
- Węglowodany: lekkie podwyższenie (+200-300 kcal)

**Niezalecane:**
- Nowe ćwiczenia (nauka wzorca = stres)
- Wysoki intensity cardio (HIIT)
- Inne intensywne aktywności

## 4. STALL-BREAKERS (Strategie na Plateau)

### Kiedy występuje plateau?
- Brak progresji przez 2-3 tygodnie (nie tylko 1 tydzień!)
- RIR stale rośnie przy tym samym obciążeniu
- Motywacja spada

### 7 Skutecznych Stall-Breakers

#### 1. Zmiana Ćwiczenia (Exercise Variation)
**Dlaczego działa:** Nowy wzorzec = nowe stimulus + nowe jednostki motorowe

**Przykłady zastępstw:**
```
Bench press z ławki poziomej → Incline bench press (15°)
Squat back → Front squat → Hack squat
Pull-ups → Chin-ups (podchwyt)
Deadlift konwencjonalny → Sumo → Trap bar
```

**Częstotliwość zmiany:** Co 4-8 tygodni na jedno ćwiczenie.

#### 2. Rest-Pause Training
**Metoda:** Jedna "mega-seria" z mini-przerwami.

**Protokół经典:**
```
Obciążenie: 70-75% 1RM (coś na 10-12 reps)
Cel: 15-20 powtórzeń łącznie w jednej serii

Wykonaj tyle reps ile możesz (np. 10)
ODPOCZNIJ 15-20 sekund
Wykonaj kolejne (np. 4)
ODPOCZNIJ 15-20 sekund
Wykonaj kolejne (np. 3)
ODPOCZNIJ 15-20 sekund
... do 15-20 reps łącznie

RIR po zakończeniu: 1-2
```

**Częstotliwość:** 1-2 razy/tydzień na partie cofnięte, nie więcej.

#### 3. Myo-Reps (popularna w Europie)
**Uproszczony rest-pause z mniej agresywnymi mini-przerwami.**

**Protokół:**
```
1. Activation set: 1 seria do upadku technicznego (lub RIR 0-1)
2. Myo-reps: 3-4 mini-serii po 2-4 reps z 10-15s przerwą

Obciążenie: typowo 65-70% 1RM

Przykład (lat pulldown):
Activation: 12 reps (RIR 0)
Rest 10s
Mini 1: 3 reps
Rest 10s
Mini 2: 3 reps
Rest 10s
Mini 3: 2 reps
Rest 10s
Mini 4: 2 reps
→ 22 reps łącznie
```

**Kluczowe:** Kiedy spadasz poniżej 2 reps w mini-serii → zwiększ obciążenie o 2.5-5%.

#### 4. Cluster Sets
**Mini-serie wewnątrz jednej serii.**

**Protokół:**
```
Obciążenie: 80-85% 1RM
Struktura: (4-6 reps) + (15-30s rest) × 3-4 clusters = 1 set
Total reps per set: 12-18 (ale każdy cluster jest submaximalny)

Przykład:
4 reps @ 85%
Rest 20s
4 reps @ 85%
Rest 20s
4 reps @ 85%
= 12 reps łącznie w 1 "cluster set"

RIR po każdym cluster: 1-2
RIR na koniec całego setu: 0-1
```

**Zastosowanie:** Siła głównie, ale hipertrofia również.

#### 5. Drop Sets (Stripping Method)
**Protokół:**
```
Set 1: Do upadku technicznego @ 80%
Redukcja obciążenia o 20-25%
Set 2: Do upadku technicznego (zazwyczaj +3-6 reps)
Redukcja obciążenia o 20-25%
Set 3: Do upadku technicznego
```

**UWAGA:** Drop sets są **HIGHLY FATIGUING**. Używaj rozważnie:
- Max 1-2 razy/tydzień
- Nie na ćwiczenia złożone (squat, deadlift)
- Idealne dla izolacji (biceps, triceps, lateral raises)

#### 6. Intra-Set Stretching
**Protokół:**
```
Między seriami tego samego ćwiczenia:
1. Skończ serię
2. Natychmiast przejdź do pozycji stretch (np. dla biceps: skłonie podparte)
3. Utrzymaj stretch przez 30-45s (nie ból!)
4. Reszta normalna (90-120s)
```

**Mechanizm:** Rozciąganie pod napięciem = mechanotransdukcja + lepsza regeneracja metaboliczna.

#### 7. Change of Tempo
**3 protokoły tempa:**

| Typ | Koncentryczne | Ekscentryczne | ISO (pause) | Zastosowanie |
|-----|---------------|---------------|-------------|--------------|
| **Standard** | 1s | 2s | 0s | Główna większość |
| **Controlled** | 1s | 3-4s | 0s | Hipertrofia |
| **Eccentric focus** | 1s | 4-6s | 0s | Hipertrofia + siła ekscentryczna |
| **Paused** | 1s | 2s | 2-3s | Siła startowa |
| **Explosive** | X (maks szybkość) | 2s | 0s | Moc (power) |

**Przykład zastosowania:**
```
Tydzień 1-2: Standard tempo
Tydzień 3-4: Eccentric focus (4-6s)
Tydzień 5-6: Paused (2s na dole)
→ Zmiana tempa = nowy stimulus bez zmiany ćwiczenia
```

## 5. MIKROCYKL TYGODNIOWY (Mikrocykl PPL)

### PUSH / PULL / LEGS - 4-6 dni/tydzień

**Opcja A: 4 dni (dla intermediate)**
```
Pon: PUSH (główna)
Wt:  PULL (główna)
Śr:  Rest lub active recovery
Czw: PUSH (secondary)
Pt:  PULL (secondary)
Sb:  LECS (full)
Nd:  Rest
```

**Opcja B: 5 dni (zaawansowani)**
```
Pon: PUSH (główna)
Wt:  PULL (główna)
Śr:  LECS (główna)
Czw: PUSH (secondary)
Pt:  PULL (secondary)
Sb:  Active recovery
Nd:  Rest
```

**Opcja C: 6 dni (elita)**
```
Pon: PUSH (główna)
Wt:  PULL (główna)
Śr:  LECS (główna)
Czw: PUSH (secondary)
Pt:  PULL (secondary)
Sb:  LECS (secondary)
Nd:  Rest
```

### Przykładowy Tydzień (Intermediate Block - Hypertrophy Focus)

#### MONDAY (PUSH - Main)
```
1. Bench press: 4x8 @ 70% (RIR 2)
   - Rest 120s

2. Overhead press: 3x10 @ 68% (RIR 2-3)
   - Rest 90s

   - Rest 90s

4. Lateral raises: 3x12-15 (RIR 1)
   - Rest 60s

5. Triceps pushdown: 3x10-12 (RIR 1)
   - Rest 60s

```

#### TUESDAY (PULL - Main)

```
   - Rest 120s

2. Bent over row: 3x10 @ 68% (RIR 2-3)
   - Rest 90s

3. Face pulls: 3x15 (RIR 1-2)
   - Rest 60s

4. Bicep curl (barbell): 3x10-12 (RIR 1)
   - Rest 60s

5. Hammer curl: 2x12-15 (RIR 1)
   - Rest 60s
```

#### WEDNESDAY (Active Recovery)

```
Option A: 20-30 min cardio @ 60-70% HRmax
Option B: Yoga/mobility session (30-45 min)
Option C: Light walk (30-45 min)
Option D: Rest day (jeśli bardzo zmęczony)

```

```
1. Dips (weighted): 3x8-10 (RIR 2)
   - Rest 90s
```

```
   - Rest 90s

   - Rest 60s
```

#### SATURDAY (LEGS - Full)
```
1. Back squat: 4x8 @ 72% (RIR 2)
   - Rest 150s

Volume: 20 setów
```

#### SUNDAY (Rest)
```
Full rest day - zero structured exercise
Focus on: sleep (8h+), nutrition, mobility (optional light stretching)
```

### Progresja w ramach tygodnia

**Strategia autoregulacji (Tygodniowa):**

```
If any session RIR > cel przez 2 tygodnie:
→ Zwiększ obciążenie o 2.5% w następnym tygodniu

If RIR < cel (np. target 2 RIR, ale masz 0 RIR):
→ Utrzymaj obciążenie + opcjonalnie -1 set w następnej sesji

If performance drop ≥10% vs previous week:
→ Redukcja o 5-10% load lub -20% volume
```

## 6. MESOCYKL 4-6 TYGODNI

### Struktura complete mesocyklu (Hypertrophy → Strength → Peaking → Deload)

**Czas trwania: 5-7 tygodni** (zależnie od poziomu)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MESOCYKL HYPERSTRENGTH                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  FAZA 1: AKUMULACJA (Tygodnie 1-2)                                   │
│  ├─ Focus: Hipertrofia + work capacity                              │
│  ├─ Volume: 15-20 setów/partia/tydzień                             │
│  ├─ Intensity: RIR 2-3 (65-75% 1RM)                                │
│  └─ Reps: 8-15                                                     │
│                                                                     │
│  FAZA 2: INTENSYFIKACJA (Tygodnie 3-4)                              │
│  ├─ Focus: Siła + przenoszenie hipertrofii                         │
│  ├─ Volume: 10-14 setów/partia/tydzień                             │
│  ├─ Intensity: RIR 1-2 (78-85% 1RM)                                │
│  └─ Reps: 4-8                                                      │
│                                                                     │
│  FAZA 3: REALIZACJA (Tygodnie 5-6)                                  │
│  ├─ Focus: Maksymalizacja wyniku                                    │
│  ├─ Volume: 6-10 setów/partia/tydzień                              │
│  ├─ Intensity: RIR 0-1 (85-95% 1RM)                                │
│  └─ Reps: 1-5                                                      │
│                                                                     │
│  FAZA 4: DELOAD (Tydzień 7)                                         │
│  ├─ Focus: Regeneracja + nadadaptacja                               │
│  ├─ Volume: 50% z fazy 3                                           │
│  ├─ Intensity: RIR 4-5                                             │
│  └─ Czas: 1 tydzień (plus dodatkowy jeśli PF > 3 na skali)         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Szczegółowy rozkład tygodniowy (Mesocykl 6-tygodniowy)

#### TABELA ZMIENNYCH

| Tydzień | Faza | Volume (sets/partia) | Intensity (%1RM) | RIR | Reps | Focus |
|---------|------|---------------------|------------------|-----|------|--------|
| 1 | Akumulacja 1 | 16-20 | 65-70% | 2-3 | 10-15 | Work capacity |
| 2 | Akumulacja 2 | 18-20 | 68-72% | 2-3 | 8-12 | Hypertrophy |
| 3 | Intensyfikacja 1 | 12-14 | 75-80% | 1-2 | 6-10 | Strength |
| 4 | Intensyfikacja 2 | 10-12 | 80-85% | 1-2 | 4-8 | Peak strength |
| 5 | Realizacja | 6-8 | 85-92% | 0-1 | 2-5 | 1RM testing |
| 6 | Deload | 3-4 (50% vol) | 70% (z fazy 5) | 4-5 | 5-8 | Regeneracja |

### Przykład: Bench Press przez Mesocykl

**Założenia:** 1RM = 100kg na początku

```
TYDZIEŃ 1 (Akumulacja 1):
Cel: 3x10 @ 70kg (70% 1RM)
RIR: 2-3

TYDZIEŃ 2 (Akumulacja 2):
Cel: 3x10 @ 72.5kg (72.5% 1RM)
RIR: 2-3

TYDZIEŃ 3 (Intensyfikacja 1):
Cel: 3x6 @ 80kg (80% 1RM)
RIR: 1-2

TYDZIEŃ 4 (Intensyfikacja 2):
Cel: 3x5 @ 82.5kg (82.5% 1RM)
RIR: 1-2

TYDZIEŃ 5 (Realizacja):
Dzień 1: 2x3 @ 87.5kg (test)
Dzień 2: 1x1 @ 92.5kg → nowy 1RM
Dzień 3: 1x2 @ 90kg (back-off)

TYDZIEŃ 6 (Deload):
2x6 @ 65kg (RIR 4-5)
Cel: regeneracja, nie progres
```

**Wynik końcowy:** 1RM z 100kg → 92.5kg (chyba że progress jest idealnie linearny)

### Transitions między fazami

**Kiedy przechodzić do następnej fazy?**

1. **Akumulacja → Intensyfikacja:**
   - Po 2-3 tygodniach akumulacji
   - Objętość maksymalna ale bez spadku siły
   - Motywacja wysoka

2. **Intensyfikacja → Realizacja:**
   - Po 2-3 tygodniach intensyfikacji
   - Siła rośnie ale objętość zaczyna obciążać
   - Gotowy na testing

3. **Realizacja → Deload:**
   - Po 1-2 tygodniach realizacji
   - Objętość jest bardzo niska
   - Znaki zmęczenia rosną

4. **Deload → Nowy Mesocykl:**
   - Po 1 tygodniu pełnego deload
   - Test 1RM (opcjonalnie) → nowy baseline
   - Rozpocznij od akumulacji z nowym 1RM

## 7. INDYWIDUALIZACJA (Poziom Zaawansowania)

### BEGINNER (0-1 lat treningu)

**Charakterystyka:**
- Neural adaptation > Hypertrophy
- Może progresować na sesję
- Niski work capacity

**Rekomendowany model: LINEAR PROGRESSION**

**Struktura:**
```
Frequency: 3-4 sesje/tydzień
Volume: 9-12 setów/partia/tydzień (nie więcej!)
Intensity: RIR 2-3 (nigdy do failure)
Focus: Technikę > Obciążenie
```

**Protokół progresji:**
```
Option A: Constant load, increasing reps
Tydzień 1: 3x8 @ 60kg
Tydzień 2: 3x9 @ 60kg
Tydzień 3: 3x10 @ 60kg
Tydzień 4: 3x8 @ 62.5kg (reset + load increase)

Option B: Simple linear (Starting Strength style)
Sesja A: 3x5 @ X kg
Sesja B: 3x5 @ (X + 2.5) kg
Sesja C: 3x5 @ (X + 5) kg
```

**Znaki przejścia do Intermediate:**
- Nie możesz dodać 1.25-2.5kg przez 2 tygodnie
- RIR stale rośnie (>3 przy tym samym obciążeniu)
- Motywacja spada, bo "stagnujesz"

---

### INTERMEDIATE (1-3 lat treningu)

**Charakterystyka:**
- Hypertrophy = Neural adaptation
- Progresja jest tygodniowa, nie sesyjna
- Potrafi czytać swoje RIR

**Rekomendowane modele:**
1. **DOUBLE PROGRESSION** (głównie)
2. **DUP** (opcjonalnie)
3. **Bloki krótkie** (4-tygodniowe)

**Struktura:**
```
Frequency: 4-5 sesje/tydzień
Volume: 12-18 setów/partia/tydzień
Intensity: RIR 1-3 (zróżnicowane w tygodniu)
Focus: Hipertrofia + Siła
```

**Protokół progresji (Double Progression):**
```
Tydzień 1-2: Akumulacja (RIR 2-3, high volume)
Tydzień 3-4: Intensyfikacja (RIR 1-2, medium volume)
Tydzień 5: Deload
```

**Kluczowa zasada:**
- Nie progresuj na sesję → progresuj na tydzień
- Jeśli jeden trening jest słabszy → normalne
- Jeśli 2-3 treningi w rzędzie są słabe → auto-reduction

**Znaki przejścia do Advanced:**
- Double progression nie działa (stagnacja)
- 4-tygodniowe bloki nie dają nowych PR
- Potrzebujesz bardziej zróżnicowanych bodźców

---

### ADVANCED (3+ lat treningu)

**Charakterystyka:**
- Neural adaptations są w większości wykończone
- Hipertrofia jest głównym celem
- Wymagają periodyzacji blokowej
- Progresja jest na mesocykle (4-6 tygodni)

**Rekomendowane modele:**
1. **BLOKOWA PERIODYZACJA** (głównie)
2. **DUB złożony** (opcjonalnie)
3. **Conjugate** (opcjonalnie)

**Struktura:**
```
Frequency: 5-6 sesje/tydzień
Volume: 18-25 setów/partia/tydzień (zmiennie)
Intensity: RIR 0-4 (zależnie od fazy)
Focus: Hipertrofia + Siła + Peaking
```

**Protokół progresji:**
```
Bloki 4-6 tygodni (zobacz sekcja 6)

Klucz: Nie każda sesja musi być PR
Cel: Minimal fatigue, maximal stimulus
```

**Specjalizacja:**
- Advanced mogą fokusować na partii (np. block dla chest)
- Inne partie są na "maintenance" (1/2 volume)

**Znaki Master (5+ lat):**
- Potrzebują 1-2 specialization blocks/rok
- Większość czasu na maintenance
- Progresja jest na makrocykle (3-6 miesięcy)

---

### DOPASOWANIE DO CELÓW

| Główny cel | Priorytet | Model polecany | Kluczowe zmienne |
|------------|-----------|----------------|-----------------|
| **Hipertrofia czysta** | Rozmiar > Siła | Block (Hypertrophy) | Volume 15-20, RIR 1-2 |
| **Siła relatywna** | Siła/bez masy | DUP + Conjugate | Velocity-based, RIR 2-3 |
| **Siła absolutna** | Maksymalna siła | Linear (early) → Block (late) | RIR 0-1, high % |
| **Rekomp (Redukcja)** | Utrzymanie + LBM | Low volume, high intensity | RIR 2-3, 50% volume |
| **Masa (Surplus)** | Hipertrofia | Block + DUP | Volume high, RIR 0-1 |

## 8. QUICK-START GUIDE (Implementacja Natychmiastowa)

### Dla użytkownika, który chce zacząć TERAZ

**Krok 1: Określ swój poziom**
```
< 1 rok treningu → BEGINNER (LP)
1-3 lata → INTERMEDIATE (Double Progression)
3+ lat → ADVANCED (Block)
```

**Krok 2: Wybierz model periodyzacji**
```
Beginner → Section 2.1 (Linear)
Intermediate → Section 2.3 (DUP) lub Double Progression
Advanced → Section 2.2 (Block)
```

**Krok 3: Zdefiniuj swój RIR celowy**
```
Deficyt kaloryczny → RIR 2-3 (przetrwanie)
Utrzymanie → RIR 1-2 (budowa)
Surplus → RIR 0-1 (ekspansja)
```

**Krok 4: Zastosuj progresję**
```
Double Progression (dla większości):
- Zakres 8-12 reps
- Zwiększaj reps między treningami
- Zwiększaj load gdy osiągasz upper bound
```

**Krok 5: Zaplanuj deload**
```
Time-based: Co 4-6 tygodni
Lub fatigue-based: Gdy znaki z sekcja 3.1 wystąpią
```

**Krok 6: Śledź progresję**
```
Zapisz:
- Load, reps, sets, RIR dla każdego ćwiczenia
- Notuj jak się czujesz (motywacja, sen, bóle)
- Oceniaj co tydzień czy progresja występuje
```

## 9. FREQUENTLY ASKED QUESTIONS

**Q: Jak często zmienić ćwiczenia?**
A: Dla beginner: 8-12 tygodni. Intermediate: 6-8 tygodni. Advanced: 4-6 tygodni. Zmieniaj gdy progresja停滞 (stagnuje) nie "dla zmiany".

**Q: RIR vs RPE - co używać?**
A: RIR jest bardziej praktyczne dla większości. RPE wymaga dużej doświadczenia. Zacznij od RIR.

**Q: Czy trening do upadku jest konieczny?**
A: Nie. RIR 0-1 jest wystarczające dla hipertrofii. Upadek techniczny (RIR 0) = higher fatigue, nie lepsze wyniki w większości przypadków.

**Q: Jak długo stosować ten sam model?**
A: Dopóki działa. Kiedy 2-3 tygodnie bez progresji → czas na zmianę.

**Q: Deload - czy go przeskoczyć?**
A: NIE. Deload jest niezbędny dla long-term progress. Przeskocz go → przetrenowanie → spadek wyników.

**Q: Ile setów na partię tygodniowo?**
A: Beginner: 9-12. Intermediate: 12-18. Advanced: 15-25 (zależnie od fazy).

**Q: Cardio i periodyzacja?**
A: Utrzymuj cardio stałe w trakcie mesocyklu. Zmieniaj tylko trening siłowy.

**Q: Czy można łączyć bloki?**
A: Dla advanced: tak (np. Hypertrophy block → Peaking block). Dla beginner/intermediate: nie, skomplikowane.
