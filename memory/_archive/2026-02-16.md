## Opcje dostępu do markdown z iPhone (przegadane)

| Opcja | Koszt | Sync | Edycja |
|-------|-------|------|--------|
| Filebrowser (self-hosted) | $0 | live | przeglądarka |
| GitHub + Working Copy | $0 / $5 | ręczny | Taio |
| Google Drive + rclone | $0 | ~15min | Drive app |

Wybrano: **GitHub** – najprostsze, działa z darmowymi apkami na iOS.

## Notatki
- Obsidian na iOS jest płatny ($5/miesiąc lub jednorazowo)
- iCloud Notes / Google Keep nie obsługują zewnętrznych plików .md
- Working Copy (free) + Taio (free) = darmowy workflow Git na iOS
