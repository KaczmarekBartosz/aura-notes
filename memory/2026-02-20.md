### Excel Encoding Issue - RESOLVED ✅
**Problem:** CSV displayed correctly on iPhone but showed "krzaki" (garbled text) in Excel on PC.
**Root Cause:** Excel on Windows with Polish locale expects:
- UTF-8 encoding with BOM (Byte Order Mark)
- Semicolon (`;`) separator instead of comma (`,`) for proper column splitting

**Solution:** Generated Excel-friendly version with:
- UTF-8 encoding with BOM
- Semicolon separator
- Filename: `gawin24_produkty_EXCEL.csv`
