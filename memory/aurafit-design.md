# Aurafit Design Checklist (Soft Minimal Premium)

Reusable 12-point checklist for dashboard/UI work:

1. bg-background for app, bg-card for cards
2. Cards: rounded-3xl + shadow-soft, no harsh borders
3. Matte finishes, no neon
4. Consistent radius (rounded-3xl; controls rounded-full/rounded-xl)
5. Bento grid, no mixed layout patterns (e.g., h-scroll inside feed)
6. Readable typography (tracking-tight), avoid tiny fonts on mobile
7. Touch targets ≥ 44px
8. Icons in colored capsules (bg-*-light + text-*)
9. Accent colors strictly by pillars (mint/orange/purple/blue)
10. Subtle animations only (no constant glow/ping/rotate)
11. Mobile-first spacing/stacking, avoid cramped layouts
12. Clear hierarchy & rhythm (header → key cards → secondary), avoid too many sections at once
